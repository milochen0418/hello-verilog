# truth table for wired AND and wired OR


## Define

|   sign        | meaning       |
| ------------- | ------------- |
|    0          | logic false 假  |
|    1          | logic true  真  |
|    x          | floating    浮接  |
|    z          | high impedance 高阻抗 |
|   |   |

|     **0**         |      0        |        0      |     0         |      0        | 
|     **1**         |      0        |        1      |     x         |      1        | 
|     **x**         |      0        |        x      |     x         |      x        | 
|     **z**         |      0        |        1      |     x         |      z        |
